# IDADM Helper

这个项目是用于帮助 2025 入学的 CUHK IDADM （沙田校区） 同学进行学业规划的网页应用。

> [!IMPORTANT]
> 相关资料是基于 2025 入学（沙田校区）的课程资料。
> 来自其他入学年份 / 校区同学请谨慎使用。如需使用，请留意会否存在差异。


> [!CAUTION]
> 本项目为个人开发项目，并非官方项目，不代表任何组织。
> 由于个人精力有限，无法确保资料的正确性和及时性，资料仅供参考，请多加核查。

### 🌟 主要特性
- 支持规划大学必修课程 (Ucore)
- 支持规划 IDADM 课程两个主修的必修课和选修课
- 支持计算是否达成课程毕业学分要求
- 支持计算是否超出学校修读学分限制
